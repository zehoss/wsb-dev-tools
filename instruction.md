## Build project

```bash
$ mvn install
```

## Run project

```bash
$ java -jar target/wsb-dev-tools-0.0.1-SNAPSHOT.jar
```

## Open application

Browse to http://localhost:8080/api/car

### Endpoints

* GET http://localhost:8080/api/car
* POST http://localhost:8080/api/car
* GET http://localhost:8080/api/car/{carId}

## Stop project

```bash
$ ctrl+c
```
